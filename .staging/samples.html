The Five Samples

One coherent forensic engagement, five deliverables.

Each sample below was generated from the same synthetic project — owner-caused RFI delays, contractor-caused crew shortages, and concurrent weather events spread across a 14-month build. The same engagement, viewed through five complementary forensic lenses.

DCMA 14-Point · GAO-16-89G · CPP Quality Overlay

Schedule Health Dashboard

Full DCMA 14-Point assessment with pass/fail per criterion, GAO 10 Best Practices scoring, CPP Schedule Quality Overlay (A–F grade across six categories), AI Executive Summary, baseline-vs-current slip register, and reproducibility manifest. The same dashboard a federal contracting officer would sign off on.

AACE MIP 3.3 · Windows Analysis

Forensic Windows Analysis Dashboard

Three-window observational/dynamic/contemporaneous as-is analysis (AACE 29R-03 §3.3) across the project life. Per-window completion shifts, attribution split (Owner/Contractor/Concurrent), critical-path duration growth, milestone slip register with responsible-party assignment.

AACE MIP 3.7/3.8 · Modeled / Subtractive

Collapsed As-Built / But-For Analysis

Independent dual-method validation per AACE 29R-03 §3.8 (Modeled / Subtractive / Single Base — Collapsed As-Built) of the windows result. Eight delay events removed from the as-built network one at a time and cumulatively; per-event but-for finish dates with cross-reference back to the MIP 3.3 windows result.
